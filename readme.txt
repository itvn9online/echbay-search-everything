== Installation ==
1. Upload `echbay-search-everything` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` admin menu in WordPress
3. Go to menu `Webgiare Plugins` menu and `Search Everything` menu in WordPress for custom setting.
